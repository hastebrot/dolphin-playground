# dolphin-playground

Playground for Dolphin Platform clients and server.

## usage

~~~console
# run server.
./gradlew runServer

# run client.
./gradlew runClient
~~~

## versions

- dolphin 1.0.0.CR5
- kotlin 1.3.11
- gradle 5.1.1
- intellij 2018.3.3
